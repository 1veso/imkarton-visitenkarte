# legal_docs/

Quelldokumente der rechtlichen Texte für imkarton.de.

## Live-Dokumente (auf der Website veröffentlicht)

Die folgenden Markdown-Dateien werden als HTML-Subpages live geschaltet:

| Markdown-Quelle | Live-URL |
|-----------------|----------|
| `IMPRESSUM.md` | https://imkarton.de/impressum.html |
| `DATENSCHUTZ.md` | https://imkarton.de/datenschutz.html |

**Sync-Hinweis:** Bei Änderungen an diesen `.md`-Dateien müssen die zugehörigen `.html`-Dateien im Repo-Root **manuell** synchronisiert werden. Es existiert kein automatischer Build-Step.

## Interne Dokumente (nicht veröffentlicht)

Die folgenden Dateien dienen als Vorhalt oder interne Beratungsempfehlung und werden **nicht** als HTML gerendert:

| Datei | Zweck |
|-------|-------|
| `COOKIE_HINWEIS.md` | Interne Beratungsempfehlung — wird relevant, sobald Cookies oder Tracking eingebunden werden |
| `VORHALT_AGB_B2C.md` | Vorhalt für künftiges B2C-Geschäft |
| `VORHALT_AGB_B2B.md` | Vorhalt für künftiges B2B-Geschäft |
| `VORHALT_Widerrufsbelehrung.md` | Vorhalt für künftiges B2C-Geschäft |

Diese Dateien werden über `robots.txt` von der Indexierung ausgeschlossen (`Disallow: /legal_docs/`).
